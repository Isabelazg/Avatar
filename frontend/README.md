# Frontend - Avatar Generator

Aplicación React para generación de avatares con IA.

## Instalación

```bash
npm install
```

## Ejecución

```bash
# Desarrollo
npm run dev

# Build para producción
npm run build

# Preview de producción
npm run preview
```

## Estructura

- `src/components/` - Componentes React
- `src/services/` - Servicios HTTP
- `src/utils/` - Constantes y utilidades
- `src/App.jsx` - Componente principal
- `src/main.jsx` - Punto de entrada

## Tecnologías

- React 18
- TailwindCSS
- Vite
- Fetch API
